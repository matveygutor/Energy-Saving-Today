Для корректной работы данного приложения необходимо создать базу данных на локальном сервере OpenServer
База данных должна иметь имя "Testingphp"
После создания базы данных необходимо импортировать скрипт "Testingphp.sql", который находится в папке "dump"
Папка с проектом должна быть расположена на сервере
После всех настроек откройте проект в Open Server Panel
